# Brand Logos

Sube aquí las imágenes de los logos de las marcas premium.

## Formato de archivos

Las imágenes deben tener el siguiente formato:
- **Formato**: PNG (recomendado) o SVG
- **Nombre**: Debe coincidir con el `id` de la marca en `src/data/brands.ts`
- **Ejemplo**: `princess.png`, `sunseeker.png`, `azimut.png`, etc.

## Recomendaciones

- Tamaño recomendado: 120x48px o proporción similar
- Fondo transparente (PNG con transparencia)
- Alta calidad para mejor visualización
- Si una imagen no está disponible, se mostrará el nombre de la marca como fallback
